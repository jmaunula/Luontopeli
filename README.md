# Luontopeli

APK: https://github.com/jmaunula/Luontopeli/releases/tag/luontopeli

## Toteutetut ominaisuudet

Kaikki vaaditut asiat arvosanaan 3 on toteutettu

- **Viikko 1 – M3 + navigaatio**
  - Toimiva navigaatio
  - Material 3 -teema
  - MVVM-kansiorakenne

- **Viikko 2 – Anturit**
  - Askelmittari toimii
  - WalkStatsCard näyttää datan

- **Viikko 3 – GPS + kartat**
  - osmdroid-kartta toimii
  - Reitti piirtyy Polyline-komponentilla

- **Viikko 4 – CameraX**
  - Kuvan ottaminen toimii
  - Kuvan tallennus Room-tietokantaan

- **Viikko 5 – ML Kit**
  - Kasvien tunnistus toimii
  - Luottamusprosentti näkyy

- **Viikko 6 – Offline-arkkitehtuuri**
  - Firebase-stubit toteutettu
  - DiscoverScreen listaa Room-datan

- **Viikko 7 – Viimeistely**
  - Splash screen
  - Adaptive icon
